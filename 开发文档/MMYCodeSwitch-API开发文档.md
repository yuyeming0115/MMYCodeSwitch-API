## дёҖгҖҒйЎ№зӣ®жҰӮиҝ°

MMYCodeSwitch-API жҳҜдёҖж¬ҫи·Ёе№іеҸ°жЎҢйқўеә”з”ЁпјҢз”ЁдәҺз®ЎзҗҶе’Ңеҝ«йҖҹеҲҮжҚў Claude Code CLI жүҖдҪҝз”Ёзҡ„ API дҫӣеә”е•Ҷй…ҚзҪ®пјҲз¬¬дёүж–№дёӯиҪ¬з«ҷгҖҒе®ҳж–№ APIгҖҒиҙҰеҸ·зҷ»еҪ•жЁЎејҸпјүгҖӮ

**ж ёеҝғзӣ®ж Үпјҡ**
- дёҖй”®еҲҮжҚўдёҚеҗҢ API дҫӣеә”е•ҶпјҢж— йңҖжүӢеҠЁзј–иҫ‘д»»дҪ•й…ҚзҪ®ж–Үд»¶
- е®Ңе…ЁдёҚз ҙеқҸ `.claude/` зӣ®еҪ•зҡ„й»ҳи®Өз»“жһ„е’Ңе…¶д»– CLI е·Ҙе…·зҡ„й…ҚзҪ®
- зӢ¬з«Ӣз®ЎзҗҶз©әй—ҙпјҢй…ҚзҪ®еҸҜеҜје…ҘеҜјеҮәеӨҮд»Ҫ
- ж”ҜжҢҒ Windows / macOS / Linux
- ж”ҜжҢҒеӨҡдёӘ Claude Code е®һдҫӢпјҲеӨҡ `CLAUDE_CONFIG_DIR` и·Ҝеҫ„пјү
- дёӯиӢұеҸҢиҜӯ UIпјҢйҡҸж—¶еҲҮжҚў

## е…ӯгҖҒAPI Key е®үе…Ёзӯ–з•Ҙ

еҠ еҜҶзӣ®ж Үпјҡ**йҳІжӯўжҲӘеӣҫ/еұҸе№•еҪ•еҲ¶ж—¶ Key жі„йңІпјҢйҳІжӯўй…ҚзҪ®ж–Үд»¶иў«зӣҙжҺҘиҜ»еҸ–**пјҢдёҚйңҖиҰҒзЎ¬д»¶зә§з»‘е®ҡгҖӮ

| йҳ¶ж®ө | еӨ„зҗҶж–№ејҸ |
|------|----------|
| иҫ“е…Ҙж—¶ | жҳҺж–ҮжҳҫзӨәпјҲж–№дҫҝж ёеҜ№зІҳиҙҙпјү |
| дҝқеӯҳеҗҺз•ҢйқўжҳҫзӨә | `sk-****...****`пјҲд»…жҳҫзӨәеүҚ4еҗҺ4дҪҚпјү |
| зј–иҫ‘ж—¶ | иҫ“е…ҘжЎҶй»ҳи®Өз©әпјҢplaceholderпјҡ`з•ҷз©әеҲҷдёҚжӣҙж–° / Leave blank to keep current` |
| жң¬ең°еӯҳеӮЁ | AES-256-GCM еҠ еҜҶпјҢеҜҶй’Ҙз”ұ App е®үиЈ…ж—¶йҡҸжңәз”ҹжҲҗе№¶еӯҳе…Ҙзі»з»ҹеҜҶй’Ҙй“ҫпјҲKeychain/Credential Managerпјү |
| еҜјеҮәеӨҮд»Ҫ | з”ЁжҲ·иҮӘи®ҫеӨҮд»ҪеҜҶз ҒдәҢж¬ЎеҠ еҜҶ Key |
| еҜје…Ҙж—¶ | йӘҢиҜҒеӨҮд»ҪеҜҶз Ғи§ЈеҜҶпјҢеҶҚз”Ёжң¬жңәеҜҶй’ҘйҮҚж–°еҠ еҜҶеӯҳеӮЁ |

## дёғгҖҒдҫӣеә”е•Ҷеӣҫж Үи§„иҢғ

дјҳе…Ҳзә§пјҡ
1. **дҫӣеә”е•Ҷе®ҳж–№еӣҫж Ү**пјҲеҶ…зҪ®еёёи§Ғдҫӣеә”е•ҶпјҡAnthropicгҖҒOpenRouterгҖҒйҳҝйҮҢдә‘зҷҫзӮјгҖҒMiniMaxгҖҒKimiгҖҒжҷәи°ұ ZaiгҖҒи…ҫи®Ҝдә‘гҖҒhow88 зӯүпјү
2. **з”ЁжҲ·дёҠдј еӣҫзүҮ**пјҲж”ҜжҢҒ PNG/SVGпјҢиҮӘеҠЁиЈҒеүӘдёәеңҶи§’ж–№еҪўпјү
3. **ж–Үеӯ—зј©еҶҷжӣҝд»Ј**пјҲеҸ–еҗҚз§°еүҚ2дёӘеӯ—з¬ҰпјҢж·ұиүІиғҢжҷҜпјҢеҰӮ `H8`гҖҒ`OR`гҖҒ`ALI`пјү

## е…«гҖҒзІҳиҙҙи§ЈжһҗиҮӘеҠЁиҜҶеҲ«

з”ЁжҲ·зІҳиҙҙдҫӣеә”е•ҶжҸҗдҫӣзҡ„й…ҚзҪ®д»Јз ҒпјҢApp е°қиҜ•жҸҗеҸ–е…ій”®еӯ—ж®өпјҡ

| иҫ“е…Ҙж јејҸ | иҜҶеҲ«и§„еҲҷ |
|----------|----------|
| JSONпјҲеҗ« `ANTHROPIC_BASE_URL`пјү | зӣҙжҺҘи§ЈжһҗеҜ№еә”еӯ—ж®ө |
| `export ANTHROPIC_BASE_URL=...` | shell еҸҳйҮҸж јејҸи§Јжһҗ |
| зәҜ URL иЎҢпјҲ`https://` ејҖеӨҙпјү | жҸҗеҸ–дёә baseUrl еҖҷйҖү |
| `sk-` ејҖеӨҙеӯ—з¬ҰдёІ | жҸҗеҸ–дёә apiKey еҖҷйҖү |
| ж— еүҚзјҖй•ҝеӯ—з¬ҰдёІпјҲ>20дҪҚпјү | дҪңдёә apiKey еҖҷйҖүпјҢжҸҗзӨәз”ЁжҲ·зЎ®и®Ө |

и§ЈжһҗеҗҺеұ•зӨәйў„и§ҲпјҢ**з”ЁжҲ·зЎ®и®ӨеҗҺжүҚдҝқеӯҳ**пјҢдёҚиҮӘеҠЁйқҷй»ҳеҶҷе…ҘгҖӮ

- Base URLпјҲеҝ…еЎ«пјҢAPI жЁЎејҸпјү
- й»ҳи®ӨжЁЎеһӢпјҲеҸҜйҖүпјү
- й…ҚзҪ®зӣ®еҪ•пјҲеҜ№еә”е®һдҫӢпјҢеҸҜйҖүпјү
- йўқеӨ–еҗҜеҠЁеҸӮж•°пјҲеҸҜйҖүпјү
- еӨҮжіЁ
- "д»Һй…ҚзҪ®д»Јз Ғи§Јжһҗ" жҢүй’®пјҲзІҳиҙҙи§Јжһҗе…ҘеҸЈпјү

### 10.3 и®ҫзҪ®йқўжқҝ

- иҜӯиЁҖеҲҮжҚўпјҡдёӯж–Ү / English
- з»Ҳз«ҜеҒҸеҘҪпјҲдјҳе…ҲдҪҝз”Ё Windows Terminal / iTerm2 зӯүпјү
