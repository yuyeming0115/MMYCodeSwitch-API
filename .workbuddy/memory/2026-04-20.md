## 新增功能：快速配置（QuickSetup）组件
- **新增文件** `src/components/QuickSetup.vue` — 两步式向导组件，内置阿里云百炼（DashScope）预设模板
- **修改文件** `src/components/AppContent.vue` — 集成「⚡ 快速配置」按钮和 QuickSetup 组件
- **修改文件** `src/i18n/zh.ts` / `en.ts` — 补充 quick_setup 国际化文案

### 功能说明
用户点击「快速配置」按钮后：
1. **Step 1** 选择平台模板（目前内置阿里云百炼）
2. **Step 2** 只需填写 API Key + 选接口协议（OpenAI/Anthropic 兼容）+ 选默认模型 → 点击「一键生成配置」自动创建 Provider
- 百炼预设包含全部 9 个模型：qwen3.6-plus, qwen3.5-plus, qwen3-max-2026-01-23, qwen3-coder-next, qwen3-coder-plus, glm-5, glm-4.7, kimi-k2.5, MiniMax-M2.5
- 支持 OpenAI 兼容和 Anthropic 兼容两种 Base URL 切换
- 配置预览面板展示即将生成的完整信息
